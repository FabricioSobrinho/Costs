# Costs
O Costs é um gerenciador de projetos feito em react que simula o consumo de uma API com auxílio do 'backend'

`O Costs é inteiramente feito em react utilizando algumas dependencias, são elas: `

*[React-Router-Dom]* -> Permite a navegação entre as páginas
*[React-Icons]* -> Oferece uma biblioteca com diversos ícones para serem utilizados no front-end do projeto
*[uuid]* -> Gera ID's de identificação para elementos, útil para a organização de informações no back-end

# Página inicial
`Na página inicial temos a apresentação da marca costs e o botão para redirecionar o usuário para a página de criação de projetos. `

# Página de gerenciamento
`Após a criação do projeto o usuário será redirecionado para a página onde são exibidos todos os projetos em vigor.`
## Página de edição
*Dentro da página de edição é possível mudar informações do projeto, além de adicionar novos elementos de custo*

# Página de contato
`Na página de contatos ficarão centralizadas as informações de contato da empresa.`

# Página sobre a empresa
`Nesta página que pode ser acessada pela navBar do projeto, é possível acessar as informações da empresa.`
